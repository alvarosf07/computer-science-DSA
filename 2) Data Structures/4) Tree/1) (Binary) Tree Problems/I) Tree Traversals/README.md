# Tree Traversals
A Tree Data Structure can be traversed in following ways:

* Depth First Search (DFS)
  * Inorder Traversal
  * Preorder Traversal
  * Postorder Traversal
  * Vertical Order Traversal
* Breadth First Search (BFS) or Level Order Traversal
* Boundary Traversal
* Diagonal Traversal
